Singly Linked List

A linked list is a linear data structure made of nodes. Each node stores a value and a reference (pointer) to the next node. Nodes are not stored contiguously like arrays.

Operation Time Complexity
Access (by index) O(n)
Search O(n)
Insert/Delete (front) O(1)
Insert/Delete (end) with tail O(1)
