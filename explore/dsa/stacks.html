Stacks (LIFO)

A stack is a linear data structure where the last item added is the first item removed. This is called LIFO (Last In, First Out). You mainly work with the top of the stack.

Operation Time Complexity
Push O(1)
Pop O(1)
Peek (Top) O(1)
Search O(n)
